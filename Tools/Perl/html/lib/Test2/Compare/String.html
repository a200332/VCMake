NAME

Test2::Compare::String - Compare two values as strings

DESCRIPTION

This is used to compare two items after they are stringified. You can also check that two strings are not equal.

Note: This will fail if the received value is undefined, it must be defined.

SOURCE

The source code repository for Test2-Suite can be found at http://github.com/Test-More/Test2-Suite/.
