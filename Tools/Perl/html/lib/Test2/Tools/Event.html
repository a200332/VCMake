NAME

Test2::Tools::Event - Tools for generating test events.

DESCRIPTION

This module provides tools for generating events quickly by bypassing the context/hub. This is particularly useful when testing other Test2 packages.

EXPORTS

$e = gen_event($TYPE)
$e = gen_event($TYPE, %FIELDS)
$e = gen_event 'Ok';
$e = gen_event Ok => ( ... )
$e = gen_event '+Test2::Event::Ok' => ( ... )

This will produce an event of the specified type. $TYPE is assumed to be shorthand for Test2::Event::$TYPE, you can prefix $TYPE with a '+' to drop the assumption.

An Test2::Util::Trace will be generated using caller(0) and will be put in the 'trace' field of your new event, unless you specified your own 'trace' field.
