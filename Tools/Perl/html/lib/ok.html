NAME

ok - Alternative to Test::More::use_ok

SYNOPSIS

    use ok 'Some::Module';

DESCRIPTION

With this module, simply change all use_ok in test scripts to use ok, and they will be executed at BEGIN time.

Please see Test::use::ok for the full description.
