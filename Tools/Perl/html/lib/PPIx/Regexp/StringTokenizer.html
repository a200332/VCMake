NAME

PPIx::Regexp::StringTokenizer - Tokenize a string literal

SYNOPSIS

 use PPIx::Regexp::Dumper;
 PPIx::Regexp::Dumper->new( 'qq{foo$bar}', parse => 'string' )
     ->print();

PPIx::Regexp::StringTokenizer is a PPIx::Regexp::Tokenizer.

PPIx::Regexp::StringTokenizer has no descendants.

DESCRIPTION

This class provides tokenization of string literals. It is deprecated in favor of the use of PPIx::QuoteLike.

METHODS

This class supports no public methods over and above those of the superclass.

SEE ALSO

PPIx::Regexp::Tokenizer.
