NAME

PPIx::Regexp::Token::Backtrack - Represent backtrack control.

SYNOPSIS

 use PPIx::Regexp::Dumper;
 PPIx::Regexp::Dumper->new( 'qr{(*ACCEPT)}smx' )
     ->print();

INHERITANCE

PPIx::Regexp::Token::Backtrack is a PPIx::Regexp::Token.

PPIx::Regexp::Token::Backtrack has no descendants.

DESCRIPTION

This class represents one of the backtrack controls.

METHODS

This class provides no public methods beyond those provided by its superclass.

arg

This method returns the backtrack control argument specified by the element. This is the text after the first colon (':'), or the empty string ('') if none was specified.

verb

This method returns the backtrack control verb represented by the element. This is the text up to but not including the first colon (':') if any. If the element specifies '' or 'F", this method will return 'MARK' or 'FAIL', respectively.
